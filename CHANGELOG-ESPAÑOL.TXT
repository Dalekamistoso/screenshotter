Limitaciones posibles y/o conocidas---------------------------------------------------------------------------------------

[x] Las rutas y comandos si superan el espacio de comspec del sistema no se ejecutaran correctamente por limitación
    del propio S.O.

[x] Las longuitud y profundidad de rutas están restringidas a las limitaciones de FAT y NTFS que son:
     No más de 256 caracteres de ruta
     No más de 8 niveles de profundidad de directorios
     No se soportan caracteres especiales ni nombres reservados (CON, LPT, : , etc....)

[x] Se aplican las limitaciones inherentes al OpenMSX (1 solo fichero por zip)

[x] No se ha testeado en Windows 10, si falla cambia powershell al simbolo de sistema estándar de Windows

[x] La aplicación debe ir en el mismo directorio que OpenMSX esto es así para mitigar las limitaciones del comspec y
    no crear rutas exageradamente largas.

[x] Si usas Windows 10 con PowerShell como sustituto del símbolo de sistema es posible que no se genere correctamente
    el fichero de listado, si es tu caso (yo uso Windows 7 y Linux Mint) cambia el shell por defecto al clásico.

[x] Si la máquina emulada por defecto no soporta determinado fichero por la configuración fallará, lo cual es lógico
    por lo que deberias apartar esas roms y aplicar otra configuracion por defecto para esas roms antes de procesar
    esos directorios por separado
